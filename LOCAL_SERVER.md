# Локальний сервер

Для звичайної роботи відкрийте:

```text
start-local.cmd
```

Вікно має залишатися відкритим. Поки воно відкрите, застосунок доступний тут:

```text
http://localhost:5173/
```

Для фонового запуску відкрийте:

```text
start-local-background.cmd
```

Щоб сервер стартував автоматично після входу у Windows, один раз відкрийте:

```text
install-local-autostart.cmd
```

Щоб вимкнути автозапуск:

```text
uninstall-local-autostart.cmd
```

Щоб зупинити локальний сервер:

```text
stop-local.cmd
```
